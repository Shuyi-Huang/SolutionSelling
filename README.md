# SolutionSelling
SolutionSelling is a foundational ASP.NET MVC web application developed to practice core MVC architecture and user authentication concepts.

The project focuses on implementing secure registration and login functionality, along with protected pages that require authenticated access.

---

## Project Overview

This application demonstrates:
- The structure of an ASP.NET MVC project
- Integration of ASP.NET Identity for authentication
- Access control for private pages

It was created as a learning and practice project to build a solid foundation for more advanced .NET web applications.

---


## Tech Stack

- ASP.NET MVC (.NET 6)
- C#
- ASP.NET Identity
- Entity Framework Core
- HTML / CSS
- Razor Views


## Development Environment

- Originally developed using Visual Studio for Mac
- !Note: Visual Studio for Mac was retired by Microsoft on August 31, 2024


## Purpose of This Project

This project was built to:
- Understand ASP.NET MVC fundamentals
- Learn authentication and authorization concepts
- Practice building protected web pages
- Prepare for larger, database-driven MVC applications

## Pages Included

- Home Page — Public landing page
- Private Page — Requires user authentication
- Register Page — User account creation
- Login Page — Secure sign-in
